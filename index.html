Iris Dataset Clustering & Evaluation

Python Pandas Scikit-learn Unsupervised Learning

An Unsupervisedlearning project focused on grouping iris flower species based on physical dimensions. I implemented K-Means clustering across multiple values of K, and used the Elbow Method & Silhouette Coefficients to determine the optimal number of clusters. The project includes an automated pipeline that generated & organized over 30 diagnostic scatter plots to visualize cluster separation & data patterns.
